# healcare_data_analysis2
📊 Healthcare Data Analysis Project

For my data analyst practice, I worked on a complete data analysis project using a healthcare dataset.

I started with data cleaning in MySQL, where I:

Created a staging table to preserve the original dataset
Removed duplicates using ROW_NUMBER()
Standardized the data with functions like TRIM(), UPDATE, and REPLACE()
Handled missing (null) values

Once the data was clean, I moved on to the analysis phase, using:

Subqueries
GROUP BY, ORDER BY
Aggregate functions such as COUNT(), AVG(), and MAX()

After completing the analysis, I used Power BI to build an interactive dashboard and visualize key insights.

This project was a great learning experience. It helped me better understand how to clean, structure, and analyze data, as well as how to present it in a clear and meaningful way.
