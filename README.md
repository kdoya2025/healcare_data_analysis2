# healcare_data_analysis2
📊 Healthcare Data Analysis Project


🔍 Objective


This project analyzes a healthcare dataset to identify patterns in patient demographics, medical conditions, and billing costs.

The goal is to uncover insights that can support better healthcare decision-making and resource allocation.



🛠️ Tools & Technologies




MySQL → Data cleaning & transformation


Power BI → Data visualization & dashboard creation





🧹 Data Cleaning Process




Created a staging table to preserve raw data


Removed duplicates using ROW_NUMBER()


Standardized text fields using TRIM() and REPLACE()


Converted date formats using STR_TO_DATE()


Checked and handled missing values





📊 Data Analysis




Performed aggregations using COUNT(), SUM(), AVG()


Analyzed patient distribution by:



Gender


Medical condition


Blood type






Evaluated billing amounts by disease and medication


Created age groups for better demographic analysis


Explored trends over time using admission dates





📈 Key Insights




Cancer and asthma generate the highest billing amounts, indicating major cost drivers


Patient admissions increased significantly after 2019, showing a rising demand for healthcare services


Medical conditions are relatively evenly distributed across genders, with slight variations


Medication usage is consistent across conditions, suggesting standardized treatment approaches
